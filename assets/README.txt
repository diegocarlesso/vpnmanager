Diretório reservado para ícones e recursos gráficos estáticos.

O ícone da aplicação é gerado em tempo de execução (utils/helpers.create_app_icon),
portanto nenhum arquivo é obrigatório aqui. Coloque um "app.ico" neste diretório
caso deseje substituir o ícone padrão gerado dinamicamente (ver README.md na raiz
do projeto para instruções de uso no build do PyInstaller).
